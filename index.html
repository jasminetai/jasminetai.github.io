About

Discord Bot

March 2022 - current

I wrote and currently maintain a Discord bot for a browser game. I used the Discord.js API to make this bot, which performs various commands that are of interest to the game's players. Some of its functions include scraping data from the game's website (such as for looking up a player's profile), storing data into a PostgresSql database (such as for linking players' accounts to Discord accounts), and building graphs with Chart.js (such as for visualizing players' profile data over time). As of May 2022, in regards to its handy stat-tracking functionality, it has... 10ish users.
